## 实用工具
### 拼音识别拆分
将带声调的拼音拆分为以国际音标表示的声母韵母。
```go
s, y, t, err := emozi.SplitPinyin("jiǒng")
if err != nil {
    panic(err)
}
fmt.Println(s, y, tone) // tɕ i̯ʊŋ 上声
```
